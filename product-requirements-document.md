# PRD - 自研系统首页看板

## 1. 产品概述

### 1.1 产品背景
自研系统首页看板是社保管理系统的核心数据展示平台，为管理人员提供全面的业务监控和数据分析功能。通过可视化的方式展示关键业务指标、流程进度和异常监控，帮助管理者快速了解系统运行状态并做出决策。

### 1.2 产品目标
- 提供实时的业务数据监控和展示
- 支持多维度的数据筛选和下钻分析
- 实现异常情况的及时预警和处理
- 提升管理效率和决策质量

### 1.3 目标用户
- 系统管理员
- 业务运营人员
- 数据分析师
- 高级管理人员

## 2. 功能需求

### 2.1 核心功能模块

#### 2.1.1 社保日历模块
**功能描述：** 展示各主体的关键时间节点信息

**具体需求：**
- 显示各主体关账日
- 显示调基时间
- 显示缴费时间节点
- 支持点击跳转至对应详情页面

**交互要求：**
- 日历形式展示，直观清晰
- 不同类型事件用不同颜色标识
- 支持月份切换查看

#### 2.1.2 关键指标卡模块
**功能描述：** 展示核心业务指标数据

**筛选功能：**
- 支持按提报方筛选
- 支持按处理时间筛选
- 支持按派单方式筛选（人工/系统自动）

**指标项目：**

1. **人员总数**
   - 显示当前系统管理的总人员数量
   - 点击跳转至"员工缴纳信息查询"页面

2. **账户数据**
   - 账户总数量
   - 鼠标悬浮显示：各城市账户数量、各账户下参保人数、公积金比例
   - 点击进入账户详情页面
   - 账户状态监控：正常户连续180天以上参保人数为0时报异常

3. **规则数据**
   - 入职、调转、调基规则数量
   - 鼠标悬浮显示：各事业线各规则数量
   - 点击进入规则详情页面
   - 规则版本平均更新周期
   - 规则状态监控：规则启用后连续180天无派单记录时报异常

4. **派单与订单管理**
   - 当月增员总数
     - 悬浮显示：系统入职增员数、人工入职增员数、系统调转增员数、人工调转增员数、补缴增员数
     - 点击对应类别进入相应页面
   - 当月减员总数
     - 悬浮显示：系统离职减员数、人工离职减员数、系统调转减员数、人工调转减员数
     - 点击对应类别进入相应页面

5. **业务效率指标**
   - 派单人工处理率
   - 异常派单原因统计
   - 订单平均处理时长
   - 异常参保率（应参保人数/实际参保人数）
   - 新员工参保及时率（入职后N天完成参保登记）

#### 2.1.3 流程进度监控模块
**功能描述：** 实时监控各业务流程的处理进度

**监控类别：**

1. **政策/账户/规则**
   - 显示政策更新进度
   - 账户配置状态
   - 规则执行情况

2. **派单监控**
   - 系统增员待人工处理
   - 系统减员待人工处理
   - 调转增员待人工处理
   - 调转减员待人工处理
   - 调基派单待人工处理
   - 稽核待人工处理

3. **订单监控**
   - 增员订单处理进度
   - 减员订单处理进度
   - 调整订单处理进度
   - 调转订单处理进度
   - 稽核订单处理进度
   - 调基订单处理进度

4. **质检监控**
   - 点击进入"员工缴纳信息查询"页面

#### 2.1.4 账户管理看板模块
**功能描述：** 展示账户相关的统计分析数据

**具体内容：**
1. **提报方人员分布环形图**
   - 提报方包括：FESCO、聚成、邦芒、广东民企、马鑫、天津HRSSC、华欣安、亚一明昊、前程、联朝
   - 以环形图形式展示各提报方的人员分布比例
   - 支持数据标签显示具体数值和百分比

#### 2.1.5 待办事项模块
**功能描述：** 展示需要处理的重要任务和提醒事项

**具体内容：**
- 系统异常提醒
- 待审核事项
- 重要截止日期提醒
- 支持任务状态更新
- 支持优先级标识

## 3. 非功能需求

### 3.1 性能要求
- 页面加载时间不超过3秒
- 数据刷新响应时间不超过2秒
- 支持并发用户数不少于100人

### 3.2 兼容性要求
- 支持主流浏览器（Chrome、Firefox、Safari、Edge）
- 支持响应式设计，适配不同屏幕尺寸
- 兼容移动端访问

### 3.3 安全要求
- 支持用户权限控制
- 数据传输加密
- 操作日志记录

### 3.4 可用性要求
- 界面简洁直观，符合用户使用习惯
- 提供操作指引和帮助文档
- 支持键盘快捷键操作

## 4. 界面设计要求

### 4.1 布局设计
- 采用三列布局：左侧指标卡、中间图表展示、右侧待办事项
- 顶部为筛选条件和标题区域
- 整体风格简洁现代，色彩搭配协调

### 4.2 交互设计
- 支持鼠标悬浮显示详细信息
- 点击跳转功能明确标识
- 数据加载时显示loading状态
- 异常状态用醒目颜色标识

### 4.3 数据可视化
- 使用合适的图表类型展示不同数据
- 支持图表交互（缩放、筛选、下钻）
- 颜色编码具有业务含义
