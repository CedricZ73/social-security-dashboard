# 自研新系统功能模块

## 1. 政策管理
1. **险种基数比例**
2. **社平工资和最低工资**
3. **数据同步**

## 2. 账户管理
1. **账户信息**（相当于旧系统中的"公司主体总数"）
2. **账户关账日**

## 3. 规则管理
1. **参保人员范围**
2. **缴纳规则**
3. **缴纳白名单**
4. **派单规则**
5. **派单白名单**

## 4. 派单管理
1. **派单管理**
   1. **派单类型**
      1. 入职增员
      2. 离职减员
      3. 调转增员
      4. 调转减员
      5. 调整
      6. 稽核补缴
   2. **派单状态**
      1. 待派单
      2. 异常派单
      3. 已派单
      4. 取消派单
      5. 全部

## 5. 订单管理
1. **订单管理**
   1. **订单类型**（一般情况下，一个派单对应社保/公积金两个订单。例：一个入职增员派单对应"社保-入职增员订单"+"公积金-入职增员订单"）
      1. 入职增员
      2. 离职减员
      3. 调转增员
      4. 调转减员
      5. 调整
      6. 稽核补缴
   2. **订单状态**
      1. 待处理
      2. 经办中
      3. 异常订单
      4. 已完成
      5. 待提单人处理
      6. 全部

## 6. 支付管理
1. **缴纳主体管理**
2. **提报方实缴账单上传**
3. **管理方实缴账单下载**
4. **支付明细**
5. **汇总请款**
6. **任务执行记录**

## 7. 数据查询
1. **员工缴纳信息查询**

## 8. 数据字典
1. **缴纳城市**
2. **提报方**

## 9. 下载中心
